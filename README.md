# AI Content Personalization Engine
This repository contains an AI Content Personalization Engine built using Salesforce, SingleStore Helios, and Groq Llama model. The project includes a pipeline covering data ingestion, SQL-based transformations, LLM-driven persona reasoning, FastAPI-based service endpoints, Salesforce writeback, and a dashboard for a personalization visualization.

The Notebooks must be run in SingleStore Helios. Use (Github Integration for Notebooks)[https://docs.singlestore.com/cloud/container-services/notebooks/github-integration-for-notebooks/] to clone this repo in your cloud.

For more details, refer to my (Medium article)[https://medium.com/@akriti.upadhyay/how-to-build-an-ai-content-personalization-engine-using-salesforce-singlestore-and-groq-llama-9f5cc11df99e].
